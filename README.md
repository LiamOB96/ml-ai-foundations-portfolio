# ml-ai-foundations-portfolio

This is the final project in the Codecademy machine learning and AI foundations skill path. The goal is to prepare a dataset for machine learning modelling. I decided to use the Premier League Player Stats dataset from Kaggle that refers to the 2019-2020 season, with a focus on preparing the data for goal scoring efficiency.

## Contents
- Jupyter notebook with exploratory data analysis and data preparation
- Clean dataset focusing on attacking efficiency metrics
